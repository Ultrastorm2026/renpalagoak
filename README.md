# 🎸 The RP’s — Página Oficial

Bienvenido al repositorio de la página web oficial de **The RP’s**, una banda de rock que combina energía, actitud y un sonido inconfundible.  
Este proyecto contiene la estructura principal del sitio, incluyendo la página de **Inicio** y la sección de **Merchandising**.

---

## 🏁 Inicio

La página de inicio presenta la esencia de The RP’s:

- Identidad visual de la banda (logo, colores y estética rock).
- Mensaje principal que define el espíritu del grupo.
- Acceso directo a las secciones clave: música, conciertos, merch y contacto.
- Enlaces a redes sociales para seguir a la banda.

El objetivo del inicio es transmitir quiénes somos y por qué sonamos como sonamos.

---

## 🛍️ Merch

La sección de Merch ofrece productos oficiales de The RP’s, diseñados para fans que quieren llevar el rock en su día a día.

Incluye:

- Camisetas con diseños exclusivos.
- Sudaderas y ropa de edición limitada.
- Accesorios como chapas, pegatinas o plectros.
- Información sobre precios, tallas, envíos y disponibilidad.

La página está pensada para ser visual, directa y fácil de usar.

---

## 📁 Estructura del proyecto

- `/index.html` — Página de inicio  
- `/merch.html` — Página de merchandising  
- `/assets/` — Imágenes, logos y recursos visuales  
- `/styles/` — Hojas de estilo  
- `/scripts/` — Funcionalidad adicional (si aplica)

---

## 🤘 Sobre The RP’s

**The RP’s** es una banda de rock formada por músicos que viven el directo, el volumen y la autenticidad.  
Este repositorio contiene el código de su web oficial, creada para conectar con fans y compartir su música con el mundo.
